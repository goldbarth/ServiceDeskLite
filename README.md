# ServiceDeskLite

A lightweight Service Desk showcase project built with:

- ASP.NET Core (.NET 10)
- Blazor (Interactive Server)
- Clean layered architecture
- EF Core
- MudBlazor

## Goals

- Demonstrate clean architecture
- Explicit workflow handling
- Proper error strategy (ProblemDetails)
- Testable domain logic

## Structure

- `src/ServiceDeskLite.Web` – Blazor UI
- `src/ServiceDeskLite.Domain` – Domain model and workflow rules
- `tests/` – Unit and integration tests
- `docs/` – Architecture decisions (planned)

## Status

Milestone M1 – Walking Skeleton in progress.
